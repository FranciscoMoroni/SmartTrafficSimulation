Projeto desenvolvido para a disciplina de POO.

## Tecnologias:
* Java 17+
* JavaFX
* Maven

## Padrões de Projeto Utilizados:
* **Strategy:** Para alternar entre ciclos de semáforos fixos e adaptativos.
* **State:** Para gerir os estados das cores dos semáforos.
* **MVC:** Separação clara entre Modelo, Vista e Controlador.
